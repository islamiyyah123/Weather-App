# Weather-App
Weather App 🌤️

Overview

This Weather App allows users to select a city from a dropdown menu and view current weather information. The app fetches weather data from the freeCodeCamp Weather API and displays details such as temperature, humidity, wind speed, and weather conditions.

Features

* Select a city from a dropdown list
* Fetch real-time weather data using an API
* Display:
    * Temperature
    * Feels-like temperature
    * Humidity
    * Wind speed
    * Wind gust
    * Weather condition
    * Location name
    * Weather icon
* Error handling for failed API requests

Technologies Used

* HTML
* JavaScript (ES6)
* Fetch API
* freeCodeCamp Weather API

How It Works

1. Select a city from the dropdown menu.
2. Click the Get Weather button.
3. The application sends a request to the weather API.
4. Weather information is displayed on the page.
